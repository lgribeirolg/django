1. Regressão Linear

A Regressão Linear é um modelo simples de aprendizado supervisionado usado para prever um valor contínuo com base em uma variável independente.

Exemplo de código Python

from sklearn.linear_model import LinearRegression
import numpy as np

# Dados de exemplo
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1, 2, 3, 4, 5])

# Modelo
modelo = LinearRegression()
modelo.fit(X, y)

# Previsão
print(modelo.predict([[6]]))  # Saída esperada: [6.0]

2. K-Means

O algoritmo K-Means é um algoritmo de agrupamento (clustering) que particiona dados em K clusters, minimizando a variação dentro de cada cluster.

Exemplo de código Python

from sklearn.cluster import KMeans
import numpy as np

# Dados de exemplo
X = np.array([[1, 2], [1.5, 1.8], [5, 8], [8, 8], [1, 0.6], [9, 11]])

# Modelo
kmeans = KMeans(n_clusters=2)
kmeans.fit(X)

# Previsão de clusters
print(kmeans.predict([[0, 0], [12, 30]]))  # Saída: [0, 1]

Executar no Google Colab

3. Algoritmos de Classificação

Os algoritmos de classificação são usados para prever uma categoria ou classe para dados de entrada. Um exemplo comum é o algoritmo de Classificação com o modelo de Árvore de Decisão.

Exemplo de código Python (Árvore de Decisão)

from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import load_iris

# Carregar o conjunto de dados Iris
iris = load_iris()
X = iris.data
y = iris.target

# Modelo
modelo = DecisionTreeClassifier()
modelo.fit(X, y)

# Previsão
print(modelo.predict([[5.1, 3.5, 1.4, 0.2]]))  # Saída esperada: [0] (Classe da flor Iris-setosa)

Executar no Google Colab

4. Árvore de Decisão

A Árvore de Decisão é um modelo de aprendizado supervisionado usado tanto para problemas de classificação quanto de regressão.

Exemplo de código Python

from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import load_iris

# Carregar o conjunto de dados Iris
iris = load_iris()
X = iris.data
y = iris.target

# Modelo
modelo = DecisionTreeClassifier()
modelo.fit(X, y)

# Previsão
print(modelo.predict([[5.1, 3.5, 1.4, 0.2]]))  # Saída esperada: [0]

Executar no Google Colab

5. KNN (K-Nearest Neighbors)

O KNN é um algoritmo de aprendizado supervisionado usado tanto para classificação quanto para regressão, baseado na proximidade dos dados.

Exemplo de código Python

from sklearn.neighbors import KNeighborsClassifier
from sklearn.datasets import load_iris

# Carregar o conjunto de dados Iris
iris = load_iris()
X = iris.data
y = iris.target

# Modelo
knn = KNeighborsClassifier(n_neighbors=3)
knn.fit(X, y)

# Previsão
print(knn.predict([[5.1, 3.5, 1.4, 0.2]]))  # Saída esperada: [0]

Executar no Google Colab

6. Random Forest

O Random Forest é um algoritmo de aprendizado supervisionado que utiliza uma coleção de árvores de decisão, onde cada árvore realiza uma previsão e o modelo seleciona a previsão mais comum.

Exemplo de código Python

from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris

# Carregar o conjunto de dados Iris
iris = load_iris()
X = iris.data
y = iris.target

# Modelo
rf = RandomForestClassifier(n_estimators=100)
rf.fit(X, y)

# Previsão
print(rf.predict([[5.1, 3.5, 1.4, 0.2]]))  # Saída esperada: [0]

Executar no Google Colab
